# Лабораторная работа №3

## Разработка простой темы WordPress

## Цель работы

Научиться создавать собственную тему WordPress, разобраться в её минимальной структуре и принципах работы шаблонов.

## Описание лабораторной работы

В рамках лабораторной работы была разработана простая пользовательская тема WordPress.  
Тема включает минимальный набор файлов, необходимых для работы WordPress, а также дополнительные шаблоны для отображения страниц, записей и архивов.

В ходе выполнения работы были изучены принципы работы системы шаблонов WordPress и структура темы.

---

## Инструкции по запуску проекта

1. Установить локальный сервер (например XAMPP, OpenServer или MAMP).
2. Установить WordPress.
3. Перейти в папку:

```

wp-content/themes

```

4. Скопировать папку темы:

```

usm-theme

```

5. Открыть административную панель WordPress:

```

[http://localhost/wordpress/wp-admin](http://localhost/wordpress/wp-admin)

```

6. Перейти в раздел:

```

Appearance → Themes

```

7. Активировать тему **USM Theme**.



## Шаг 4. Файл функций

В папке темы был создан файл:

```

```

Он используется для подключения стилей темы.

```php
function usm_theme_styles(){
    wp_enqueue_style('usm-style', get_stylesheet_uri());
}

add_action('wp_enqueue_scripts', 'usm_theme_styles');
```

# Примеры использования темы

После активации темы WordPress автоматически использует различные шаблоны:

- **index.php** — главная страница
- **single.php** — страница записи
- **page.php** — страницы сайта
- **archive.php** — архивы записей

# Ответы на контрольные вопросы

### Какие два файла являются обязательными для любой темы WordPress?

Обязательные файлы:

```
style.css
index.php
```

---

### Как подключаются общие части шаблонов?

Через функции WordPress:

```php
get_header();
get_footer();
get_sidebar();
```

---

### Чем отличаются index.php, single.php и page.php?

- **index.php** — основной шаблон темы
- **single.php** — отображение одной записи
- **page.php** — отображение страниц сайта

---

### Зачем нужен файл functions.php в теме?

Файл `functions.php` используется для добавления функциональности темы:

- подключение CSS и JS
- регистрация меню
- регистрация виджетов
- добавление функций темы
